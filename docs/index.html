Organizza-Foto

Ordina automaticamente le tue foto per Anno / Mese / Eventi — versione alpha.

Funzionalità principali

  • Ordinamento per Anno/Mese/Giorno usando EXIF
  • Modalità dry-run e modalità reale
  • Creazione anteprime eventi

Scarica

Scarica l'eseguibile Windows dalla release su GitHub.

Scarica .exe (ultima release)

Come usare

  1. Decomprimi/posiziona le foto in una cartella.
  2. Apri il tool e fai prima un dry-run.
  3. Se OK, esegui in modalità move per spostare i file.
